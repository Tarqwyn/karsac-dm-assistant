# Docs

This folder holds the operating docs for the project.

## Layout

- `adr/` - architecture decisions
- `rfc/` - reviewable proposals and lifecycle contracts
- `tasks/` - implementation backlogs and work items
- `design-docs/` - design and refactor notes
- top-level docs - workflows, UX, architecture, and pipeline logs
