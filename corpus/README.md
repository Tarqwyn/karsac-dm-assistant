# Corpus

This folder holds the Karsac campaign source material and generated support data used by the registry runtime.

## Layout

- `collections/` - Markdown canon, rules, monsters, entity cards, and other runtime source material
- `rules-data/` - structured JSON lookup data used by rules and encounter flows
- `reports/` - generated reports copied out of the runtime and corpus tooling
- `adversary-corpus/` - auxiliary adversary source material retained as campaign support data

Internal collection folder names are intentionally unchanged so the registry and tests keep their current assumptions.
