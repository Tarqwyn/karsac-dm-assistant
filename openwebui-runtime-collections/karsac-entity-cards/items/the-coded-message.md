# The Coded Message

**Canon File ID:** `items/the-coded-message`

**Entity Card ID:** `entity-cards/items/the-coded-message`

**Entity Type:** Item — document

**Retrieval Summary:** A coded message carried by Erwing from a dying Cobalt Soul monk to Serris. Decoded: "Get word to the Cleric. I am discovered, trust no one, continue the work." Serris reads it and goes quiet — a different quiet from the Maw quiet. She does not yet know who the Cleric is. The Cleric is Erik Mathr (DM-only).

**Use For:** Identifying the coded message and routing to the Erik Mathr resistance thread.

**Primary Detail File:** `items/the-coded-message`

**Do Not Confuse With:** Brynja's Ledger (the cargo record document; distinct), Aldric's Letter (the letter of authority; distinct).

- Erwing — the carrier.
- Serris — the recipient.
- Erik Mathr — "the Cleric" named in it.
