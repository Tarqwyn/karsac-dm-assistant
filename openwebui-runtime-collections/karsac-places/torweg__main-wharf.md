# Main Wharf, Törweg

**Canon File ID:** `places/torweg/main-wharf`
**Retrieval Summary:** The main wharf; the half-mast flag is the first wrong note on arrival.

> The main wharf; the half-mast flag visible from inbound vessels; the first wrong note.

## Overview

The main wharf runs the length of the harbour below the cliff face. In normal conditions it would be crowded with working figures, nets, catches, and noise. In Chapter 2, it has three figures and silence. The half-mast flag at the harbourmaster's tower is visible from the Greyback before the ship is alongside the dock.

Nobody on the inbound vessel knows for whom the flag is lowered.

## Description

The main wharf is the first environment-level tell that something is wrong in Törweg. Not what is wrong. Just that it is wrong — the number of people, the absence of the usual working noise, the flag. Players arrive at the wharf and the wrongness registers before they have language for it.

The three figures on the wharf — Brynja, a second Lösweg figure, and the man in the dark coat — are watching. Not unloading. Watching.
