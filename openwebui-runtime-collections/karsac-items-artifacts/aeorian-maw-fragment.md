# The Aeorian Maw Fragment

**Canon File ID:** `items/aeorian-maw-fragment`
**Retrieval Summary:** Aeorian note missed by Vane's people inside the Maw; same hand as the scroll case margin; 'It is not a weapon. It is a condition. Available to be woken or to be held.' Handout H12.

> Aeorian note inside the Maw; same hand as the scroll case margin; after two thousand years. Handout H12.

## Overview

A note on a fragment of worked stone, inside the Maw fracture, missed by Vane's housecarls when they recovered the artefacts. The hand is Aeorian-period. Serris, reading it, recognises it immediately:

*"It is not a weapon. It is a condition. Available to be woken or to be held."*

Then: *"This is the same hand as the scroll case margin. The same hand. After two thousand years."*

She goes very quiet.

This is the player-facing cosmological revelation of Thread B: the Maw and the scroll case are connected to the same intelligence, the same moment of understanding, the same deliberate placement. The binding is not random. It was made. Someone knew what they were making.

Post as handout H12 when a PC picks up the note inside the Maw.
