# Karsac Table Ruling Principles

**Rule ID:** `rules/house/table-ruling-principles`

**Retrieval Summary:** Karsac table rulings start from SRD 5.1, keep house layers separate, and label unresolved fiction-dependent outcomes as DM calls.

## Rule Summary

Karsac uses D&D 5e 2014 as the base ruleset. House interpretation should explain the base rule first, then state any Karsac layer separately.

## Procedure

1. Identify the base SRD 5.1 rule that governs the situation.
2. State that rule cleanly before adding any Karsac material.
3. Keep item-specific, character-specific, and house-rule text out of core SRD answers unless it actually changes the ruling.
4. If the source material is silent, make a DM call and label it as a DM call rather than presenting it as written rules.
5. Do not import 2024 rules or non-SRD 2014 content into the core layer.

## At the Table

Use this when an answer needs to distinguish between what 5e says, what Karsac changes, and what is still a judgment call.

## Common Questions

### When should a Karsac rule override the SRD file?

Only when Karsac-authored material clearly adds or changes the ruling. Otherwise, use the SRD rule by itself.

### What should happen when the fiction matters more than the written text?

Give the base rule, explain the fiction-sensitive edge case, and finish with a DM call.
