# Karsac Mechanical Language

**Rule ID:** `rules/house/karsac-mechanical-language`

**Retrieval Summary:** Karsac rules text uses explicit wording for permission, prohibition, recharge, duration, and unresolved DM judgment.

## Rule Summary

Karsac rule writing should be compact and unambiguous, with a clear split between written effect and DM interpretation.

## Procedure

1. Use `can` for permission and `cannot` for prohibition.
2. Use `must` only for compulsory outcomes.
3. State timing and duration directly: `until the start of your next turn`, `once per long rest`, and similar fixed phrases.
4. Use `DM call` when the source does not lock down the exact outcome.
5. Keep retrieval summaries plain and table-facing rather than literary.

## At the Table

Use this when drafting or revising Karsac rules so item text, house text, and profile answers stay consistent.

## Common Questions

### When should a rule say DM call?

When current source material does not define the exact mechanical outcome and the table must decide from context.

### Should flavour replace procedure text?

No. Flavour can support the rule, but the procedure still needs to be explicit.
