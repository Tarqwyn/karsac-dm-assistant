# Player-Safe Rulings

**Rule ID:** `rules/house/player-safe-rulings`

**Retrieval Summary:** Player-safe Karsac rulings should answer the rules question directly without exposing DM-only campaign facts unless the table has already revealed them.

## Rule Summary

Rules help should stay useful to players without leaking hidden campaign information.

## Procedure

1. Give the base 5e rule first.
2. Add a Karsac table rule only if it is already player-facing or explicitly safe to share.
3. Leave out DM-only item states, unrevealed lore, hidden triggers, and secret consequences.
4. If a full answer depends on hidden information, say that the final outcome is a DM call.

## At the Table

Use this when answering a player-facing rules question from the shared profile or when preparing text that should stay spoiler-safe.

## Common Questions

### Can a player-safe answer mention Karsac divergences?

Yes, if the divergence is already known at the table or is safe to reveal as a standing rule.

### What if the rule question is entangled with a hidden artefact state?

Answer the base rule, give any player-visible modifier, and leave the hidden part in the DM layer.
