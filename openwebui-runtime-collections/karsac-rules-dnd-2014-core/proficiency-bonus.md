# Proficiency Bonus

**Rule ID:** `rules/core/proficiency-bonus`

**Retrieval Summary:** The proficiency bonus applies to checks, saves, and attacks only when a rule says the creature is proficient.

## Rule Summary

Characters gain a proficiency bonus by level, and monsters have one built into their stat blocks. It applies only when proficiency is granted.

## Procedure

1. Check whether the creature is proficient with the save, skill, tool, or attack being used.
2. Add the proficiency bonus once if proficiency applies.
3. If a rule doubles or halves the proficiency bonus, adjust it once before adding it.
4. If the creature is not proficient, do not add it.

## At the Table

Use this whenever a player asks whether proficiency applies or whether multiple features can add it again.

## Common Questions

### Can proficiency be added twice?

No. A single roll or number gets the proficiency bonus only once unless a rule changes the size of that one bonus.

### What if a feature doubles a proficiency I do not have?

It does nothing. Doubling or halving a proficiency bonus does not create proficiency.
