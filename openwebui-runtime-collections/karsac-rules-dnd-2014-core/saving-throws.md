# Saving Throws

**Rule ID:** `rules/core/saving-throws`

**Retrieval Summary:** Saving throws are d20 rolls used to resist harmful effects such as spells, traps, poison, and disease.

## Rule Summary

A saving throw uses an ability modifier and sometimes proficiency to resist an effect that puts a creature at risk.

## Procedure

1. An effect calls for a saving throw and sets its own DC.
2. Use the named ability.
3. Roll a d20 and add the relevant ability modifier.
4. Add proficiency if the creature is proficient in that save.
5. Apply advantage, disadvantage, and other modifiers.
6. Compare the total to the DC and apply the effect's success or failure text.

## At the Table

Use this when a creature resists danger rather than actively attempting a task.

## Common Questions

### Who chooses the save DC?

The effect that causes the save sets the DC. Spell save DCs come from the caster's spellcasting ability and proficiency bonus.

### Does a successful save always cancel the effect?

No. A successful save does whatever the effect says, which is often no effect or reduced effect.
