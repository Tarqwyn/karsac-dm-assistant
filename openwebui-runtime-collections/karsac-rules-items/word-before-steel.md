# Word Before Steel

**Rule ID:** `rules/items/word-before-steel`

**Retrieval Summary:** Word Before Steel is a known starting property of Ragnfrid's arm-ring: once per long rest, it adds 1d4 to a roll.

## Rule Summary

Current Karsac source text gives this property a simple mechanical statement and leaves finer timing and edge handling to table adjudication.

## Procedure

1. Confirm the arm-ring is present and this property has not been used since the last long rest.
2. Add `1d4` to the roll.
3. Mark the property expended until the next long rest.
4. If the exact timing or eligible roll is disputed beyond the current source text, resolve that as a DM call.

## At the Table

Use this when the ring's witness-and-verse theme matters and the table wants the known mechanical part without opening the full growth document.

## Common Questions

### Is there more detailed timing in current source material?

Not in the currently indexed item summary. The known mechanical text is `once per long rest, +1d4 to a roll`.

### Does this replace the normal roll?

No. It modifies the roll by adding `1d4`.
